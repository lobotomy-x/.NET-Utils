# .NET-Utils
single file/namespace of reusable static classes for various application tasks, e.g. github release downloader, process management, shortcut creation without COM

Useful low dependency helpers made while working on UEVR frontend upgrades that would be easy to adapt to pretty much any C# app

Comments are good enough for now...

also see [wpf-notify](https://github.com/lobotomy-x/WPF-NotifyIcon-Minimal) and [uevr-frontend](https://github.com/lobotomy-x/uevr-frontend) (source not uploaded at time of posting)
